# RomBot Community Skill

Ask the AI Agents community brain a question from inside your coding agent.

## Install

1. Get a token from Rom (DM RomBot on WhatsApp).
2. `python3 skills/rombot-community/scripts/rombot-ask.py setup`
3. `/rombot-community <question>` in your coding agent.

## Supported harnesses

Claude Code, Codex, Pi, Cursor, and any Agent Skills-compatible harness.
